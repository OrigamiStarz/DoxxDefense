# Secret Scribbles

## Requirements
Run the following commands:
1. `npm i express`
2. `npm i bootstrap`
3. `npm i scss`
4. `npm i body-parser`
5. `npm i express-fileupload`
6. `npm i dotenv`

Convenient for testing:
1. `sass --watch style.scss public/style.css`
2. `nodemon server.js`
3. make sure to have your env file
